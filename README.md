# MKV to mp4 converter in the browser
Stack:
  - React
  - Tailwind
  - Nodejs
  - Ffmpeg
  - Cluster
  - Axios
  - Cors
  - Multer
  - Nodemon
  - EXpress

# how do I launch it?
  - Copy repo for you idle
  - use ```npm run dev``` in the ``frontend`` folder
  - use ```npm start``` in the ``server`` folder

``Successful conversion! The conversion speed depends on your computer !``
